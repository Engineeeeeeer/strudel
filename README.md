# Strudel Scripts

A collection of small scripts, experiments, and utilities for use with [Strudel](https://strudel.cc/).

## Overview

This repository contains various Strudel snippets, ranging from simple pattern experiments to more advanced control structures.

## Usage

* Open Strudel
* Copy a script from this repository
* Paste and run it in your environment

Some scripts may require additional setup (e.g. WebMIDI).
